Tipos de puerta de enlace

En el modelado de procesos de negocio con BPMN, existen varios tipos de puerta de enlace para diferentes tipos de comportamiento de control, como tomar decisiones, bifurcar, fusionar, bifurcar y unir.

 

Inclusivo
Se puede utilizar una puerta de enlace inclusiva para modelar rutas alternativas y paralelas dentro de un proceso. A diferencia de la puerta de enlace exclusiva, se evalúan todas las expresiones de condición. Se tomarán todos los caminos de salida que den un resultado positivo de evaluación.

Evento intermedio

Un evento intermedio indica dónde sucede algo entre el evento de inicio y el final de un proceso. Puede usar un evento intermedio para mostrar dónde se reciben o envían los mensajes, mostrar el retraso necesario, realizar el manejo de excepciones y mostrar la necesidad de compensación. Puede colocar un intermedio incluso en dos lugares: adjuntando el límite de la tarea / subproceso, flujo normal (es decir, conectado desde un flujo sin adjuntar a una actividad).
